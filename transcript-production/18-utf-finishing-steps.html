UTF Finishing Steps

1. Full Margin Justification

  1. Position cursor at the beginning of the body and press Ctrl + Shift + End
  2. Press assigned keys to justify margin (Ctrl + Shift + J for me)
Where should I justify margins?

Applied to the body of the transcript (from "Proceedings taken in the...")

Not applied to the title page or index

2. Bookmarks in PDF

  1. Use UTF Index Template with Bookmarks (automatically creates bookmarks in PDF file)
  2. Modify bookmarks
    1. Add TITLE PAGE, TABLE OF CONTENTS, and EXHIBITS bookmarks
    2. Modify witness names to appear in all caps
    3. Modify and reposition exhibit bookmarks so they're listed chronologically

3. Initial View Settings

  1. On PDF, File menu > Properties
  2. Open the Initial View tab
  3. Set Navigation Tab to Bookmarks Panel and Page
  4. Set Page Layout to Single Page Continuous
  5. Set Magnification to Fit Width
  6. Set Open to Page to 1 (default)
  7. Click OK
This needs to be done every time because Adobe doesn't remember these settings
