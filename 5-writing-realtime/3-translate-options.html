Translate Options

Drag/Drop Analysis

Case CAT analyzes untranslates that contain characters you tend to drag or characters that are missing that you tend to drop in order to find a match to a steno outline in a dictionary being used to translate the file.

If Case CAT finds a match, the translated text will be surrounded by Drag/Drop Being and End format symbols.

For example, you tend to drag your final L. "Grate" is in your Personal Dictionary. You write GRAEULT. The untranslate will appear and "grate."

Specify the characters that you tend to drag and/or drop in Drag/Drop Analysis Details.

While editing, if Drag/Drop Analysis has translated the untranslate correctly, you can press Enter to confirm the translation.

To enable: Tools menu in Manage Jobs screen, Options, Translate, Translate, Input tab, select Drag/Drop Analysis, select Details, add keys you tend to drag and drop by clicking on the keyboard diagram (e.g., -L is commonly dragged)

TrueStroke Drag/Drop Analysis
If using:

  • Luminex via USB/Wi-Fi connection
  • Diamante via USB/Wi-Fi connection
  • Wave via USB connection
    Cast CAT analyzes untranslates during realtime translation using TrueStroke data located in the Steno X-subfile to match a steno outline in a dictionary being used to translate.

TrueStroke Drag/Drop has priority over Drag/Drop Analysis if both options are selected.

While editing, if TrueStroke Drag/Drop has translated the untranslate correctly, you can press Enter to confirm the translation.

To enable: Tools menu in Manage Jobs screen, Options, Translate, Translate, Input tab, select TrueStroke Drag/Drop Analysis. No other action required.

Suffix Drag Phase Maker

When active, Case CAT translate, by default, steno outlines accidentally stroked with a dragged letter -G, -S, or -D as a suffix letter tucked inside a steno outline.

Suffix Drag/Phrase Maker allows you to select whether dragged steno should be considered a suffix or a word.

Suffix Drag
Suffix Drag applies the suffix strokes -G, -S, -Z, or -D to single-stroke untranslates when the dragged letter is the last character in the outline.

The translated text will be surrounded by Suffix Drag Begin and End format symbols.

For example, "blog" is defined in your Personal Dictionary, but the steno stroke BLOGD is not. The untranslate will appear as "blogged."

Set Suffix Drag Options by clicking the Details button. Use -S, -Z, or both for a plural suffix. Tuckable suffixes are permitted. For example, the untranslated SLAOEGT would translate as "sleeting."

Phrase Maker
For example, let's say the steno stroke TPH-GT is not defined in the dictionary. However, you have TPH defined as "in" and -GT is set as a "word" type in your Suffix Drag/Phrase Maker Options as the word "this."

With the Word option in Suffix Drag/Phrase Maker, you can have TPH-GT translate as "in this" even though it is not defined in your dictionary as a single stroke.

Suffix Drag/Phrase Maker only applies to one-stroke words in which steno characters can be added as a suffix. Suffix Drag/Phrase Maker options allow you to add other steno characters you want Case CAT to consider a suffix.

The Plural Endings Options allow you to use the final S, final Z, or both for plural endings. To confirm the combined words are spelled correctly, set the Suffix Drag/Phrase Maker Spell Check option for the steno outline.

If Spell Check is active, Case CAT checks the newly combined word to confirm the combined word is spelled correctly. If a correctly spelled word is not found, Case CAT tries the next possible suffix to find an actual word. This may result in a correctly spelled word but not the word intended. If Case CAT does not find a correctly spelled word with an added suffix, the steno outline remains an untranslate and no conversion takes place. The Spellchecked option has no influence on phrases created by Suffix Drag/Phrase Maker.

If also using TrueStroke Drag/Drop or Drag/Drop Analysis, TrueStroke Drag/Drop and then Drag/Drop Analysis takes priority.

To enable: Tools menu in Manage Jobs screen, Options, Translate, Input tab, select Suffix Drag/Phrase Maker, select Details, add a new entry by clicking on the blank row at the bottom of the list

Some common phrasing:

Delete Repeating Q/A/C, Use Smart Quotes and Smart Parentheses

Delete Repeated Q/A/C
If multiple Qs, As, or Colloquy paragraphs (with the same name) are written on right after another with no text in between, Case CAT will automatically delete the empty paragraphs so only one paragraph displays.

For example, if you write the steno outline defined as <Question> five times in a row without text, only one Q will display in the transcript.

Use Smart Quotes
By selecting this option, you only need one dictionary definition for a quotation mark. When you write the steno defined as a quotation mark (") the first time in a paragraph, Case CAT inserts an opening quote. When you write that same steno outline again, Case CAT inserts a closing quote.

If you wrote the steno defined as quotation mark once and did not write it again to close the quotation, Case CAT automatically puts a closing quote at the end of the current paragraph style

For CACR students, this option should be selected.

Use Smart Parentheses
By selecting this option, you only need one dictionary definition for a parenthesis or bracket. When you write the steno defined as the open parenthesis or bracket the first time, Case CAT inserts an open parenthesis or bracket. When you write the same steno outline a second time, Case CAT inserts a closed parenthesis or bracket.

If you wrote the steno defined as open parenthesis or bracket once but did not write it again, Case CAT automatically inserts the closed parenthesis or bracket at the end of the current paragraph.
