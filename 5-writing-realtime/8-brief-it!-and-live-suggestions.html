Brief It! and Live Suggestions

Intro

What does it do?
A realtime feature that suggests conflict-free brief forms for words and phrases as you are writing. When you write three or more steno strokes, Case CAT generates a brief form or reminds you of an existing brief form and displays it in the Brief It dialog pane.

For example?
You have written "personal chequing account" twice. Case CAT suggests P*K as a brief, which appears in the Brief It pane on a yellow background. You like the suggestions and begin writing P*K, and it translates as "personal chequing account."

You write "joint account" out. Case CAT reminds you that J-KT is a brief in your Personal Dictionary. The reminder appears in the Brief It pane on a green background.

How do I do it?
Enabled Brief It during realtime translation:

  • Press Ctrl t to open the Translate dialog box and press Alt p (Options).
  • Click the Realtime tab.
  • In the Realtime Aids section, select Brief It.
  • In a realtime file, open the Brief It pane by pressing Alt v, e, t (View, Panes, Brief It). When you want to use a suggested brief, simply write the steno.
Brief Types Purpose
Suggested briefs generated for words/phrases written with three/more steno strokes
Reminder briefs existing dictionary entries with fewer steno strokes than what was written
Personal briefs list entries not in a dictionary (holding tank for briefs you want to try out)

Live Suggestions Into

What does it do?
A realtime feature that automatically offers translation suggestions when an untranslate displays in the transcript. You have the option to ignore, replace, or define the suggestion from the writer during realtime or with a keyboard command.

For example?
You misstroke and write SHRAOUR. In the Brief It pane, "sure" is the first and correct suggestion. You write a realtime command from your writer to replace the untranslate.

How do I do it?
Enable Live Suggestions during realtime translation:

  • Press Ctrl t to to open the Translate dialog box and press Alt p (Options).
  • Click the Realtime tab.
  • In the Realtime Aids section, select Live Suggestions.
    Add the Live Suggestions realtime commands to your Personal Dictionary.
  • In System Files, open the Realtime Commands Dictionary.
  • Select each of the {Replace With LS#} and {Define With LS#} entries. Right mouse click and copy to your Personal Dictionary.
  • In your Personal Dictionary, modify the steno as desired. If you select to display more than three Live Suggestions in your options, create new dictionary entries and LS4 and LS5.
    In a realtime file, open the Brief It pane by pressing Alt v, e, t (View, Panes, Brief It).

Write the steno stroke for the {Replace With LS#} or {Define With LS#} command that corresponds to the correct Live Suggestion.

Tips and Tricks
At least four translated words or phrases need to be written after the untranslate before a Live Suggestion will be offered.

Only one Live Suggestion is offered at a time. The suggestions display until the next untranslate is written.

Live Suggestions appear on a light blue background in the Brief It pane.

One you select a Live Suggestion, the background color turns gray and the remaining suggestions appear crossed out. To hide suggestions that were not selected, deselect the Rejected option in the Brief It pane filter options.

You can also right click a Live Suggestion in the Brief It pane and select Replace with this Suggestion or Define this Suggestion. Double clicking a Live Suggestion will replace the untranslate.

Configure Brief It & Live Suggestions

Modify Brief It Options

  • From the All Options dialog box, under Edit Panes, select Brief It.
  • From the Brief It pane, click the Options icon.
  • From Manage Jobs, press Alt t, o, e, b (Tools, Options, Edit, Brief It).
  • From Edit, press Alt t, o, Enter, b (Tools, Options, Pane Options, Brief It).

Show Phonetic Steno
Check this box to have Brief It display steno as it sounds. For example, the suggested brief for "Yoko Ono" would appear as Y*O as opposed to KWR*O.

When Show Phonetic Steno is selected, steno in the Cat Scratch Text/Steno section will also display phonetically.

Only Suggest 1 Stroke Briefs
Check this box to have Brief It suggest only one-stroke Suggested briefs. By default, Brief It always suggests a one-stroke brief with the asterisk before suggesting a two-stroke brief.

Selecting this option also affects Reminder and Personal briefs. For example, you write "neurotoxicity" in four or five strokes. When this option is active, Brief It does not remind you that you have TPHAOUR/TOX in your Personal Dictionary as a two-stroke entry.

Write Twice to Define
Brief It, by default, automatically creates a J-Define or a K-Define the first time you use a Suggested brief. Checking this box requires you to consecutively write a brief twice before Brief It creates a J-Define or K-Define. You may want to use this option to try a Suggested brief before making it a definition.

For example, Brief It suggests A*BT for "Abbott Labs." If the Write twice to Define option is active, you would need to write A*BT/A*BT to have Case CAT create the global. Once defined, you write A*BT for "Abbott Labs" for the remainder of the job.

Define to Case Dictionary
Check this box to place a Suggested brief in the case dictionary once it is written. If the current job is not in a case, then the brief is placed in the job dictionary.

Personal/Reminders Only
Check this box to display only Reminder and Personal briefs in the Brief It dialog pane. Suggested briefs are not created.

1 Stroke Reminders for 2 Strokes
Check this box to have Brief It offer one-stroke Reminders or Personal briefs when writing two-stroke entries. For example, if you write "I don't" in two strokes and there is a one-stroke version in your Personal Dictionary, Brief It shows the one-stroke brief as a Reminder.

Auto Open Brief It
Check this box to automatically display the Brief It dialog pane when starting realtime. By default, this option is selected.

Send Used Briefs to Cat Scratch
Check this box to automatically send a Suggested brief you use to the Steno/Text section of the Cat Scratch dialog pane.

Steno Include/Exclude
Displays the Brief It Options table where you can modify the Require and Exclude entries.

The Require entry contains the steno character or steno combination to be included in every Suggested brief. There is only one Require entry.

Exclude entries contain steno characters or steno combinations to exclude from every Suggested brief. You can have multiple Exclude entries.

Personal Briefs
Personal Briefs are entries which are created prior to translation and are stored in the Briefs.txt file in the System Files case. A default Briefs.txt file exists for each user.

Creating entries in the Briefs.txt file allows you to store words and phrases along with the brief form without placing them in a dictionary. For example, you got some good ideas for briefs from an article in the JCR but you're not certain you want to put them in your Personal Dictionary.

Common Words
The CommonWords.txt file contains a list of words the Brief It does not consider when looking for words and phrases for brief suggestions. Is a phrase is written that begins or ends with a word from the CommonWords.txt file, no brief is suggested. For example, by default the word "about" is in the CommonWords.txt file. When a phrase is written that begins or ends with the word "about" and no Reminder or Personal brief exists for the word or phrase, Brief It does not suggest a brief.

Allow Briefs with Numbers
Check this box to have Brief It offer suggestions for repeated dollar amounts or addresses.

An additional option is available in the Steno Include/Exclude list where a Required for Numbers entry is available. For example if the Required for Numbers entry is EU and you write 23925 several times throughout the transcript, Brief It might suggest 23EU9. (EU is the default steno entry.)

Be default, the following number/steno key combinations are excluded in the Brief Include/Exclude Options: #1S, #2K, #3W, #4R, #6-R, #7-B, #8-G, #9-S

To prevent any number bar number from being suggested, press Ctrl + i and type # and press Enter.

Allow Double First Syllable
Check this box to have Brief It offer the first syllable of the text (single word or phrase) written twice as the suggested brief. For example, the word "fluorescent" is used multiple times in the transcript. You write "fluorescent" in multiple strokes and Brief It suggests TPHROR/TPHROR.

When you have both Write Twice to Define and Allow Double First Syllable options selected, if the suggested brief has two identical strokes, Brief It will behave as if "Write Twice to Define" option was not active. In other words, you do not need to write the brief four times to use it. If the brief is one stroke or is multiple (not identical) strokes, briefs will need to be written twice to be used.

No Nag List
Check this box to designate words or phrases that should no longer be offered as a suggestion or a reminder. For example, you have the phrase "of the" in your Personal Dictionary, but sometimes you write it in two strokes. You do not want Brief It to remind you of the phrase.

From the Brief It pane, right click an entry or highlight multiple entries and then right click. Select Send Text to No Nag List. Alternatively, select No Nag List in Brief It options and type the text portion of the entry.

Suggested Configurations

Insert Oops when replacing with a live suggestion
Select to add an <Oops> format symbol (category 10) whenever you select to replace an untranslate with a Live Suggestion. This enables you to scan to Live Suggestion replacements and confirm change. You can also view each instance in the Hotspots pane.

Number of Suggestions
Select how many suggestions should be offered in the Brief It pane. Select from 2 to 5 suggestions. The default number of suggestions is 3.

Dictionary Defines go to
Select the destination dictionary you want a defined suggestion to go to when using the {Define With LS#} commands. When this option is set to case dictionary and the transcript is not in a case, entries will go to the job dictionary.

Show "translated" text as first Live Suggestions
By default, this option is active and Drag/Drop, Suffix Drag, and phonetically translated text will be the first choice in the Live Suggestion list.
Live Suggestions favor: When?
Quality For faster computers. Live Suggestions will spend a little more time looking for a suggestion and will search more steno outlines for a longer match.
Balanced Default setting.
Speed For slower computers. Live Suggestions will take minimal time searching for a suggestion and will search fewer steno outlines for a longer match.
