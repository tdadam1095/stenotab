Remote Realtime Reporting

CaseViewNet Intro

  • Send realtime texts to attorneys, judges, and litigants in the same room through a Wi-Fi hotspot or ethernet cable connection (no cables, Bluetooth, or other hardware installations)
  • CaseViewNet Cloud allows you to send your realtime text to any client with an internet connections through an encrypted, secure feed
  • Compatible with any CAT software (all editing changes are instantly updated on your clients' displays with Case CAT)
  • You have complete control over which client/s can see the transcript
  • Full line of viewing options
    • CaseViewNet for Windows for desktop users
      • Can create marks, annotations, custom issue codes, and search with dynamic indexing
      • Can save transcript and annotations in a variety of litigation software formats
        • .SGCVN
        • .PTF
        • .TXT
        • .XMPTF
    • ICVNet for IPad users
      • Can create marks and search
    • CaseViewNet Browser edition
      • Clients log on to www.caseviewnet.com to view your realtime feed using the Cloud Session Code and password
      • Solution available for any device with a web browser (phones, tablets, and computers), no software installations needed
      • Has transcript and CART-viewing options
      • Can create marks and search
      • Can use Google Translate to translate the text displayed into other languages (works well for bilingual CART clients)

To prep settings for realtime application:
Translate & Edit > Output to: CaseView (or Multiple Outputs i.e. in addition to CaseView if providing remote CART who have to send to a third party program to go directly into a feed for captioning) > View of Change CVNET Passwords > New Password - generates new line of information to use immediately of change if you want > Allow Save means to allow user to save transcript at the end of the session (file is in CaseViewNet format - must use PC version to review or file/save as whatever format they want the product to be or wait for you to convert it for them BUT they won't have any of their notes/highlighted lines) > Expiration means password expiration (don't have to get a new code every single day for e.g. a multi-week job) > Browser Link copies the session code and password to the clipboard and generates a link with the PC version with everything auto-filled, paste this in the chat app you're using (e.g. Zoom) and all clients have to do is follow it, (add their name unless you've also configured this), and click "Connect"

CaseViewNet Summary

CaseViewNet

Use a router to create a secure LAN (local area network) to wirelessly send realtime text to parties in the same room.

Reporter

  • CaseViewNet license – purchased from Stenograph

Client

  • computer with CaseViewNet for Windows installed (full litigation support features)
  • iPad with iCVNet app (limited features – marking and searching)

Features

  • Rapid Refresh to instantly update the entire realtime transcript
  • Control over viewer rights to the transcript
